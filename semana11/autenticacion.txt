El login es el proceso mediante el cual un usuario se identifica para acceder 
a una aplicación. Esto permite controlar quién puede ingresar y qué acciones 
puede realizar. Algunas rutas deben estar protegidas porque contienen 
información o funciones de uso exclusivo para usuarios autorizados. 
De esta manera se evita el acceso no autorizado y se protege la información 
del sistema. No se deben exponer públicamente contraseñas, documentos de 
identidad, teléfonos, direcciones, tokens, claves de API ni otros datos 
sensibles.

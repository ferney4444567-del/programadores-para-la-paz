Repositorio
    Un repositorio es un espacio donde se almacenan los archivos y
     el historial de cambios de un proyecto. Permite organizar y
    controlar las modificaciones realizadas por los desarrolladores.

Commit
    Un commit es un registro permanente de los cambios realizados
     en un proyecto. Cada commit guarda una versión específica 
     para poder consultar o recuperar el trabajo realizado.

Push
    El comando push envía los commits realizados localmente hacia un 
    repositorio remoto como GitHub. Esto permite respaldar el trabajo 
    y compartirlo con otros desarrolladores.

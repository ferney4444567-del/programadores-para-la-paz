Sistema operativo:
Es el software principal del equipo. Administra los recursos del computador 
y permite que los demás programas funcionen correctamente.

Node.js:
Es un entorno que permite ejecutar código JavaScript fuera del navegador. 
Se utiliza para crear servidores y aplicaciones web.

Aplicación server.js:
Es el archivo que contiene el código del servidor. Define las rutas y las 
respuestas que recibirán los usuarios.

Puerto:
Es el canal por donde el servidor recibe solicitudes. En esta práctica se 
utiliza el puerto 3000 para acceder al servicio.

Logs:
Son registros que guardan información sobre la actividad del sistema. 
Ayudan a identificar errores y monitorear el funcionamiento del servidor.
